# FreeServiceIndex

Índice de servicios gratuitos con una interfaz minimalista y rápida.

## Stack

- Vite
- React
- Oat UI
- Framer Motion

## Librerías instaladas

### Dependencias (runtime)

- @knadh/oat — ^0.4.2 — UI base en CSS/JS (estilos globales y componentes semánticos).
- framer-motion — ^11.18.2 — animaciones y transiciones en la UI.
- lucide-react — ^0.446.0 — iconos SVG como componentes React.
- react — ^18.3.1 — núcleo de la UI.
- react-dom — ^18.3.1 — renderizado en el DOM.
- react-markdown — ^10.1.0 — render de contenido Markdown en React.
- react-router-dom — ^7.6.2 — enrutado del lado del cliente.
- remark-gfm — ^4.0.1 — soporte de Markdown con GitHub Flavored Markdown.
